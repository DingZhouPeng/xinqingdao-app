# 心晴岛 APP 运行说明

## 🚀 快速启动

### 1. 安装依赖
```bash
npm install
```

### 2. 开发模式运行
```bash
npm run dev
```
访问：http://localhost:5173

### 3. 生产构建
```bash
npm run build
```
构建产物在 `dist/` 目录

### 4. 预览生产版本
```bash
npm run preview
```

## 📱 运行哪个文件？

**开发模式**：
```bash
npm run dev
```
然后在浏览器打开 http://localhost:5173

**生产模式**：
```bash
npm run build
npm run preview
```
然后在浏览器打开显示的地址（通常是 http://localhost:4173）

**直接打开 HTML**：
构建后可以直接打开 `dist/index.html`，但可能遇到跨域问题。建议使用 `npm run preview`。

## 📦 打包为 Android APP（可选）

### 1. 确保已完成构建
```bash
npm run build
```

### 2. 添加 Android 平台
```bash
npx cap add android
```

### 3. 同步代码到 Android
```bash
npx cap sync android
```

### 4. 在 Android Studio 中打开
```bash
npx cap open android
```

然后在 Android Studio 中构建 APK。

## 🎯 主流程演示路径

1. **首次进入**：填写昵称、困扰、支持人 → 进入小岛
2. **记录心情**：首页点击"记录心情" → 选择"紧张、8分、学习考试"
3. **ABC 引导**：填写事件、想法、情绪行为
4. **生成行动卡**：获得体、智、劳、美四类任务
5. **完成后评分**：从 8 分降到 5 分 → 保存
6. **查看成长**：底部导航"成长" → 看到记录和统计
7. **岛屿变化**：返回首页，岛屿阳光值、水滴、小灯增加

## 💡 常见问题

**Q: 数据存在哪里？**
A: 使用浏览器 LocalStorage，数据仅保存在本地。

**Q: 如何重置数据？**
A: 打开浏览器开发者工具 → Application → Local Storage → 删除 `xinqingdao-state-v1`

**Q: 社交功能是真实的吗？**
A: 社交数据是模拟的，不会连接真实服务器。这是为了比赛演示设计。

**Q: 能在手机上运行吗？**
A: 可以！使用 `npm run dev`，然后在手机浏览器访问电脑的局域网 IP:5173
