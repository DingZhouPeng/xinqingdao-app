# 00 产品简报

## 名称

《心晴岛 APP：校园情绪补给与五育成长伙伴》

## 一句话定位

当学生感到烦、慌、难过、生气或没动力时，心晴岛可以在 30 秒内帮助他看见情绪，在 3 分钟内给出一个可执行的小行动，并通过岛屿养成、技能卡和同伴温暖让他愿意持续使用。

## 目标用户

- 小学高段、初中、高中学生。
- 重点场景：考试压力、人际矛盾、亲子沟通、作业拖延、自我否定、睡前烦躁。

## 产品主张

```text
先好用，再讲心理学。
先让学生愿意打开，再让心理学发挥作用。
先解决“我现在怎么办”，再解释“为什么有效”。
```

## 核心闭环

```text
有情绪 → 快速记录 → 即时补给 → 小行动 → 再评分 → 岛屿成长 → 同伴温暖 → 下次再用
```

## 非目标

- 不做心理疾病诊断。
- 不做真实陌生人社交平台。
- 不做开放匿名树洞。
- 不做医疗咨询或危机干预替代品。
