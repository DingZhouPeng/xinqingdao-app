# 01 PRD：产品需求文档

## 1. 页面清单

| 页面 | 用户任务 | 产品目标 | 心理学底层 |
|---|---|---|---|
| Onboarding | 设置昵称、困扰、支持人 | 个性化与安全准备 | 社会支持 |
| 首页心晴岛 | 看今日状态、快速入口 | 抓注意力与回访 | 正向反馈 |
| 心情天气 | 选择情绪、强度、来源 | 情绪觉察 | 情绪命名、量化 |
| 晴晴助手 | 用 ABC 梳理事件想法结果 | 把混乱变清楚 | ABC 模型、认知重评 |
| 心晴行动卡 | 完成一个五育小任务 | 立即可执行 | 行为激活、自我效能 |
| 情绪急救包 | 60 秒呼吸/暂停/启动 | 快速稳定 | 放松训练、正念稳定 |
| 温暖小队 | 收到贴纸、漂流瓶、小队树 | 弱社交支持 | 社会支持 |
| 我的心晴册 | 看记录和变化 | 成长可见 | 自我观察、反思 |
| 求助灯塔 | 找支持人，生成求助话术 | 安全出口 | 求助意识 |

## 2. MVP 主流程

```text
Home → Mood → AI Guide → Action Card → After rating → Growth
```

## 3. 用户故事

### 考试紧张

作为学生，我明天考试前很慌，我想先稳定下来，然后知道现在能复习什么。

验收：选择“紧张 + 强度 8 + 学习考试”后，系统推荐呼吸、复习拆解、整理桌面。

### 和朋友吵架

作为学生，我和同学闹矛盾后很生气，我想知道怎样表达不伤人。

验收：选择“生气 + 同伴关系”后，系统推荐暂停 90 秒和“我感到……我希望……”句式。

### 没动力

作为学生，我作业很多但不想动，我希望 APP 给我一个很小的启动动作。

验收：选择“没动力/烦躁 + 作业压力”后，系统推荐 3 分钟启动任务。

### 强烈低落

作为学生，如果我输入高风险表达，APP 不应该继续普通建议，而要提醒我联系真实的人。

验收：风险词命中时跳转或突出展示求助灯塔。

## 4. 功能优先级

P0：首页、心情记录、ABC、行动卡、再评分、成长记录、急救呼吸、求助灯塔。

P1：岛屿装扮、温暖贴纸、漂流瓶、小队树、技能卡。

P2：班级能量树、更多动画、H5 分享卡、老师后台原型。
