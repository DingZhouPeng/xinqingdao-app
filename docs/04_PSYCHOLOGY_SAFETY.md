# 04 心理学与安全边界

## 1. APP 使用的心理学原理

| 功能 | 原理 | 用户看到的体验 |
|---|---|---|
| 心情天气 | 情绪命名 | 选择今天像什么天气 |
| 强度条 | 情绪量化 | 拖动云朵给情绪打分 |
| 晴晴助手 | ABC 模型 | 把事情、想法、结果分开 |
| 换个角度 | 认知重评 | 生成更具体、可行动的新想法 |
| 呼吸圆圈 | 放松训练 | 跟着圆圈吸气呼气 |
| 三分钟启动 | 行为激活 | 做一个很小的行动 |
| 温暖贴纸 | 社会支持 | 收到别人一盏小灯 |
| 成长记录 | 自我效能 | 看见自己能调节 |
| 求助灯塔 | 求助意识 | 有困难时找真实的人 |

## 2. 文案边界

应该说：

- “你现在可能有比较强的紧张情绪。”
- “我们先做一个让身体稳定的小动作。”
- “一次考试不能代表全部能力。”
- “如果你觉得撑不住，请联系身边可信任的大人。”

不应该说：

- “你有焦虑症。”
- “你抑郁了。”
- “你心理有问题。”
- “不用告诉家长老师。”
- “APP 可以治好你。”

## 3. 风险词处理

命中风险词或强度 9-10 时：

1. 不再输出普通聊天式建议。
2. 显示安全提示。
3. 引导联系支持人、老师、家长、当地紧急救助渠道。
4. 可同时给短时稳定动作：坐下、放下危险物、联系真实的人、让身边有人陪伴。

## 4. 社交安全

MVP 不做：

- 陌生人自由评论。
- 陌生人私信。
- 匿名树洞长文。
- 热度排行。

MVP 做：

- 预设贴纸。
- 模板漂流瓶。
- 小队共同任务。
- 班级匿名能量树。
