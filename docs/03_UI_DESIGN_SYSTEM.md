# 03 UI 设计系统

## 1. 视觉方向

目标：让学生觉得“这是我的小岛”，而不是“我在做心理测评”。

关键词：心情天气、岛屿养成、校园温暖、软卡片、轻游戏化、低压力。

## 2. 色彩 Token

| Token | 色值 | 用途 |
|---|---:|---|
| `--sky-50` | `#eef9ff` | 背景雾白 |
| `--sky-100` | `#d9f1ff` | 浅天空 |
| `--sky-300` | `#8ecfff` | 主按钮、重点 |
| `--mint-200` | `#a8efd5` | 成长、岛屿 |
| `--sun-200` | `#ffe27a` | 阳光值、奖励 |
| `--coral-300` | `#ff9d8a` | 情绪提醒、行动 |
| `--lavender-200` | `#d9c8ff` | 社交温暖 |
| `--ink-900` | `#223044` | 正文 |
| `--muted` | `#75839a` | 辅助文字 |

## 3. 字体与排版

- 页面标题：22-26px，700，圆润。
- 卡片标题：16-18px，700。
- 正文：14-15px，line-height 1.7。
- 按钮：15px，600。

## 4. 组件风格

### 大卡片

- 圆角 28px。
- 背景白色 75%-90% 不透明。
- 阴影柔和，避免沉重。
- 边框使用 1px 半透明白。

### 主按钮

- 渐变：天空蓝 → 薄荷绿。
- 高度 48-56px。
- 圆角 18-22px。
- 按下时轻微缩小。

### 情绪 Chip

- 使用天气 emoji + 情绪词。
- 选中后出现发光边框和轻微上浮。

### 行动卡

- 像抽卡一样展示，但不做强刺激抽奖。
- 顶部有卡牌名，中部是解释，底部是五育任务。

## 5. 关键视觉效果

### 岛屿场景

用 CSS 生成：天空渐变、云、太阳、小岛、树、灯塔、水面。

状态变化：

| 用户行为 | 视觉反馈 |
|---|---|
| 完成记录 | 云散开一点 |
| 完成行动卡 | 小树长大、水滴增加 |
| 收到鼓励 | 岛上亮一盏灯 |
| 强情绪但完成求助 | 灯塔发光 |

### 呼吸动画

圆圈跟随呼吸：

```text
吸气：圆圈变大
停顿：圆圈发光
呼气：圆圈变小
```

### 社交反馈

不要做消息瀑布流。使用贴纸、灯、贝壳、水滴等轻反馈。

## 6. 移动端规范

- 根容器最大宽度 430px。
- 桌面端显示手机壳效果。
- 底部导航固定，避开安全区。
- 页面主内容底部留 96px，避免被导航遮挡。
- 所有可点区域高度不小于 44px。

## 7. 禁止的 UI

- 医院风、问卷风、冷冰冰表格。
- 红色警报大面积铺满，除非求助灯塔高风险提示。
- 成绩排行、情绪排行、焦虑榜。
- 让用户感觉“我心理有病”的标签。
