## 3. UI 验收

- 375px 宽度无横向滚动。
- 底部导航不遮挡按钮。
- 首页岛屿视觉明确。
- 呼吸动画可见。
- 行动卡五育任务可读。
- 求助灯塔入口明显。

- 能截图放进说明文档。
