#### ✅ 4. 音效系统
**实现内容：**
- Web Audio API 生成音效（无需外部文件）
- 5种音效：
  - `click` - 按钮点击（800Hz 正弦波）
  - `pop` - 气泡消除（600Hz 正弦波）
  - `coin` - 获得金币（1000→1200Hz 方波）
  - `complete` - 完成任务（880→1100Hz 双音）
  - `tap` - 呼吸游戏点击（700Hz 短促）
- 音效集成：
  - 消消乐：每次消除气泡播放 `pop`
  - 呼吸游戏：每次点击播放 `tap`
  - 完成流程：播放 `complete`
  - 获得金币：播放 `coin`
  - 所有按钮点击：播放 `click`
- 音效设置面板：
  - 音效开关
  - 背景音乐开关（预留）
  - 音量滑块（0-100%）
  - 设置持久化（localStorage）
- 浮动设置按钮（右下角，60fps 悬浮动画）

**文件：**
- `src/services/audio.ts` - 音效管理器
- `src/components/AudioSettings.tsx` - 设置面板
- `src/styles/audio-settings.css` - 样式
